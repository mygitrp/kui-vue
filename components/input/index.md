### Input API
| 属性       | 说明                                       | 类型             | 默认值 |
|------------|--------------------------------------------|------------------|--------|
| value      | 绑定的值，可使用 `v-model` 双向绑定        | String 、 Number | -      |
| size       | 按钮尺寸,可选值 `small`、`large`，默认不选 | String           | -      |
| icon       | 输入框图标                                 | String           | -      |
| icon-align | 输入框图标位置，可选值为 `left`、`right`   | String           | right  |
| icon-click | `icon` 的点击事件                          | Function         | -      |
