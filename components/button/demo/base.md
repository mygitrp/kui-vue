<cn>
#### 基本用法
使用type、hollow、circle属性来定义Button 的样式。
</cn>

```tpl
<template>
  <div>
    <Button>Default</Button>
    <Button type="primary">Primary</Button>
    <Button type="primary" hollow>hollow</Button>
    <Button type="primary" circle>circle</Button> 
    <Button type="link">link</Button>
  </div>
</template>
```
