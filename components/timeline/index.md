### API
| 属性    | 说明            | 类型     | 默认值 |
|-------|---------------|--------|-----|
| icon  | 时间轴 `item` 图标 | String | -   |
| color | 时间轴 `item` 图标颜色 | String | -   |
| dot   | 自定义时间轴点       | slot   | -   |
| mode| 通过设置 `mode` 可以改变时间轴和内容的相对位置	`left` \| `alternate` \| `right`	|String|left|
