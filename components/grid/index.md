### Row API
| 属性     | 说明              | 类型            | 默认值 |
|--------|-----------------|---------------|-----|
| gutter | 栅格间距，单位 px，左右平分 | String,Number | 0   |
### Col API
| 属性     | 说明                    | 类型            | 默认值 |
|--------|-----------------------|---------------|-----|
| span   | 栅格的占位格数，可选值为0~24的整数   | String,Number | -   |
| offset | 栅格左侧的间隔格数，可选值为1~24的整数 | String,Number | -   |
