### Space API
| 属性      | 说明     | 类型                          | 默认值     |
| --------- | -------- | ----------------------------- | ---------- |
| align     | 对齐方式 | start , end ,center ,baseline | center     |
| direction | 间距方向 | vertical,horizontal           | horizontal |
| size      | 间距大小 | small,middle,large,number     | small      |
