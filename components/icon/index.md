### API 
| 属性    | 说明           | 类型            | 默认值   |
|-------|--------------|---------------|-------|
| type  | 图标的名称        | String        | -     |
| size  | 图标的大小，单位是 px | String,Number | -     |
| color | 图标的颜色        | String        | -     |
| spin  | 图标是否选择       | Boolen        | false |
