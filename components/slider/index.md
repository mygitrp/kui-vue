### Slider API
| 属性   | 说明                                                                       | 类型                                            | 默认值 |
| ------ | -------------------------------------------------------------------------- | ----------------------------------------------- | ------ |
| min    | 最小值                                                                     | number                                          | 0      |
| max    | 最大值                                                                     | number                                          | 100    |
| step   | 间距大小                                                                   | 步长，取值必须大于 0，并且可被 (max - min) 整除 | 1      |
| change | 当 Slider 的值发生改变时，会触发 change 事件，并把改变后的值作为参数传入。 | Function                                        | -      |
